# Plano de Pesquisa em Veículos Oficiais e Produção de Conteúdo

## Visão Geral

Este documento estabelece o fluxo de trabalho para pesquisa em veículos oficiais, produção de conteúdo e estratégias de SEO e backlinks sem investimento financeiro.

---

## Parâmetros do Projeto

| Parâmetro | Valor |
|-----------|-------|
| **Equipe** | Todos os usuários cadastrados no admin do CIN |
| **Orçamento** | R$ 0 (somente ferramentas gratuitas) |
| **Volume** | 5 artigos por dia |
| **Meta** | Maximizar retenção e revenue AdSense |

---

## 1. Priorização de Categorias

Baseado em dados do código fonte (alto CPC/RPM) e práticas de mercado:

### 🎯 Ordem de Prioridade

| # | Categoria | Justificativa |
|---|-----------|---------------|
| 1 | **Economia / Política Monetária** | Maior CPC (Fed, BCB, juros), alto valor por clique |
| 2 | **Criptomoedas** | CPC altíssimo (R$ 15-50/clique) |
| 3 | **Geopolítica** | Alta retenção, leitores engajados |
| 4 | **Tecnologia** | Volume, boas taxas de clique |
| 5 | **Energia e Commodities** | Assuntos recorrentes, IPC favorável |

## 2. Fluxo de Produção Colaborativo

```
┌─────────────────────────────────────────────────────────────────┐
│                    FLUXO DE PRODUÇÃO COLABORATIVO               │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌──────────┐    ┌──────────┐    ┌──────────┐                │
│   │ Redator  │───►│ Revisor  │───►│ Editor   │                │
│   │ (cria)   │    │ (verifica)│    │ (publica)│                │
│   └──────────┘    └──────────┘    └──────────┘                │
│        │               │               │                        │
│        ▼               ▼               ▼                        │
│   ┌─────────────────────────────────────────────┐               │
│   │         TODOS OS USUÁRIOS ADMIN             │               │
│   │  (cada um pode assumir papel conforme       │               │
│   │   disponibilidade - sistema flexível)        │               │
│   └─────────────────────────────────────────────┘               │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Papéis e Responsabilidades

| Papel | Responsabilidade | Pode Executar |
|-------|-----------------|---------------|
| **Redator** | Pesquisar fontes, redigir artigo, adicionar dados e estatísticas | Todos os usuários |
| **Revisor** | Verificar fatos, confirmar fontes, validar informações | Editores |
| **Editor** | Aprovar, publicar, gerenciar agendamento | Administradores |

**Nota**: Uma mesma pessoa pode exercer múltiplos papéis.

## 7. Estratégia de Backlinks (Sem Investimento)

### Métodos Gratuitos

| Método | Descrição | Prazo |
|--------|-----------|--------|
| **Google Discover** | Criar conteúdo de qualidade para aparecer | Longo |
| **Redes Sociais** | Compartilhar organicamente | Curto |
| **Parcerias** | Acordos com outros sites | Médio |
| **Diretórios BR** | Submit em diretórios nacionais | Curto |
| **Guest Post** | Escrever para outros blogs (sem pago) | Médio |

### Boas Práticas para Backlinks

| Fazer | Evitar |
|-------|--------|
| Links de sites relevantes | Links de farms |
| Âncora natural | Over-optimization |
| Links Dofollow | Links Nofollow apenas |
| Diversidade de domínios | Concentração excessiva |
| Links contextuais | Links em footer |

## 8. Checklist de Publicação

### Pré-Publicação

- [ ] Título com 50-60 caracteres
- [ ] Slug otimizado
- [ ] Categoria correta
- [ ] Excerpt com 150-160 caracteres
- [ ] Conteúdo com mínimo 1000 caracteres
- [ ] Imagem de capa (1200x630px)
- [ ] Autor selecionado
- [ ] Tags configuradas (mín 3)
- [ ] Verificação de links

### Publicação

- [ ] Modo: Publicar agora ou agendar
- [ ] Destacar na home (se relevante)
- [ ] Marcar urgente (se breaking)
