# Google Alerts - Configuração Rápida

## O que é o Google Alerts?

O Google Alerts envia notificações por email quando aparecem novos resultados relevantes para suas pesquisas.

## Como Configurar

1. Acesse: https://www.google.com/alerts
2. Faça login com sua conta Google
3. Adicione os alertas abaixo
4. Configure para receber "No máximo uma vez por dia"

## Dicas de Uso

### Operadores de Busca

| Operador | Exemplo | Resultado |
|----------|---------|-----------|
| Aspas | "taxa de juros" | Resultados com frase exata |
| OR | Bitcoin OR Ethereum | Um OU outro |
| - | economia -brasil | Exclui termo |
| site: | site:reuters.com | Apenas esse site |

### Boas Práticas

1. **Comece com poucos alertas** - 5-10 no máximo
2. **Ajuste conforme necessidade** - Se receber muito spam, refine a busca
3. **Use aspas** para frases específicas
4. **Monitore** os resultados e取消 os que não são relevantes

## Automatização (Opcional)

### IFTTT (Gratuito)

Você pode usar IFTTT para:
- Enviar alertas para Slack/Telegram
- Postar automaticamente em redes sociais
- Salvar em planilhas

### Zapier (Versão Gratuita)

- Mesmo 기능 que IFTTT
- Limite de 100 tarefas/mês na versão free
